# 🌱 PlantMind AI

## AI Powered Plant Disease Detection

PlantMind AI is a deep learning based web application that detects whether a plant leaf is Healthy or Diseased using image classification.

## Features

- Upload plant leaf image
- AI based disease detection
- Confidence score
- Plant care recommendations
- Clean Streamlit interface

## Technologies Used

- Python
- TensorFlow
- Streamlit
- NumPy
- Pillow

## How to Run

1. Install requirements

pip install -r requirements.txt

2. Run the application

streamlit run app.py
