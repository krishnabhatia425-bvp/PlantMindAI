# 🌱 PlantMind AI - Plant Disease Detection System

PlantMind AI is an AI-based plant disease detection system that uses Deep Learning to identify whether a plant leaf is healthy or diseased.

Users can upload a plant leaf image and get prediction results instantly.

## 🚀 Features
- Plant leaf disease detection using Deep Learning
- Upload leaf images for prediction
- CNN-based trained model
- Real-time prediction using Streamlit

## 🛠️ Tech Stack
- Python
- TensorFlow / Keras
- Streamlit
- NumPy
- Pillow

## 📂 Project Structure
PlantMindAI/ │ ├── app.py ├── model/ ├── dataset/ ├── requirements.txt └── README.md

## ⚙️ How to Run Locally
1. Clone the repository

git clone https://github.com/krishnabhatia425-bvp/PlantMindAI.git

2. Install dependencies

pip install -r requirements.txt

3. Run the application

streamlit run app.py

## 🎯 Working

1. User uploads a plant leaf image.
2. Image is preprocessed.
3. Deep Learning model analyzes the image.
4. System predicts whether the leaf is healthy or diseased.
